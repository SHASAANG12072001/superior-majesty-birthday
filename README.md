# Happy Birthday My Superior Majesty

A special animated birthday greeting card website created with HTML, CSS, and JavaScript.

## Features
- **Interactive Gift Box:** Click to open and reveal the surprises.
- **Animated Surprises:** Smooth transitions, floating hearts, and particle effects.
- **Background Music:** Includes a floating play/pause button.
- **Responsive Design:** Optimized for both mobile and desktop viewing.

## Project Structure
- `index.html`: The main entry point containing the animation and logic.
- `assets/`: Directory containing images (`queen.png`) and audio (`music.mp3`).

## How to Run
Simply open `index.html` in any modern web browser.
